          Carrier credentials — password vault

          Store login credentials per carrier so you don't have to remember them across devices. Passwords are AES-256-GCM encrypted at rest with a key in .secrets/secrets.key (gitignored). The system does not auto-fill them — you copy/paste manually when logging in. The dashboard never shows a stored password unless you click Reveal.
